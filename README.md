# OOP Python ChatRoom

A beginner-friendly chatRoom project in Python demonstrating Object-Oriented Programming (OOP). Users can join a chatroom, send messages, and save chat history.

## Features

- Users can join or leave a chatroom
- Broadcast messages to everyone
- Private messaging
- Message history with timestamps
- Search messages by keyword
- Save chat history to a file
- Demonstrates OOP concepts: classes, objects, methods

## How to Run

1. Make sure Python 3.x is installed.
2. Open terminal/command prompt in project folder.
3. Run:

```bash
python chatroom.py
